# AI-Based Motor Fault Monitoring and Predictive Maintenance System

## Overview

This project presents an AI-based predictive maintenance system for industrial motors using the ESP32 microcontroller, IoT sensors, MQTT communication, and a real-time monitoring dashboard. The system continuously acquires sensor data, processes important motor health parameters, and provides early fault detection to minimize unexpected equipment failures.

## Objectives

* Monitor motor operating conditions in real time.
* Detect abnormal motor behaviour using sensor data.
* Transmit data wirelessly using MQTT.
* Visualize motor parameters through a dashboard.
* Demonstrate the concept of predictive maintenance using IoT.

## Features

* Real-time motor monitoring
* ESP32-based embedded system
* MQTT communication
* Wokwi online simulation
* Live dashboard visualization
* Sensor data acquisition
* Early fault indication
